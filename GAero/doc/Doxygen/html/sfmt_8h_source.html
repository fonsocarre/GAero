14 * Description:
15 * This header file contains class declarations and other definitions for the
16 * "SIMD-oriented Fast Mersenne Twister" (SFMT) random number generator.
17 *
18 * The SFMT random number generator is a modification of the Mersenne Twister
19 * with improved randomness and speed, adapted to the SSE2 instruction set.
20 * The SFMT was invented by Mutsuo Saito and Makoto Matsumoto.
21 * The present C++ implementation is by Agner Fog.
22 *
23 * Class description:
24 * ==================
25 * class CRandomSFMT:
26 * Random number generator of type SIMD-oriented Fast Mersenne Twister.
27 *
28 * Member functions (methods):
29 * ===========================
30 * Constructor CRandomSFMT(int seed, int IncludeMother = 1):
31 * The seed can be any integer.
32 * Executing a program twice with the same seed will give the same sequence of
33 * random numbers. A different seed will give a different sequence.
34 * If IncludeMother is 1 then the output of the SFMT generator is combined
35 * with the output of the Mother-Of-All generator. The combined output has an
36 * excellent randomness that has passed very stringent tests for randomness.
37 * If IncludeMother is 0 then the SFMT generator is used alone.
38 *
39 * void RandomInit(int seed);
40 * Re-initializes the random number generator with a new seed.
41 *
42 * void RandomInitByArray(int seeds[], int NumSeeds);
43 * Use this function if you want to initialize with a seed with more than
44 * 32 bits. All bits in the seeds[] array will influence the sequence of
45 * random numbers generated. NumSeeds is the number of entries in the seeds[]
46 * array.
47 *
48 * double Random();
49 * Gives a floating point random number in the interval 0 <= x < 1.
50 * The resolution is 52 bits.
51 *
52 * int IRandom(int min, int max);
53 * Gives an integer random number in the interval min <= x <= max.
54 * (max-min < MAXINT).
55 * The precision is 2^(-32) (defined as the difference in frequency between
56 * possible output values). The frequencies are exact if (max-min+1) is a
57 * power of 2.
58 *
59 * int IRandomX(int min, int max);
60 * Same as IRandom, but exact. The frequencies of all output values are
61 * exactly the same for an infinitely long sequence.
62 *
63 * uint32_t BRandom();
64 * Gives 32 random bits.
65 *
66 *
67 * Example:
68 * ========
69 * The file EX-RAN.CPP contains an example of how to generate random numbers.
70 *
71 * Library version:
72 * ================
73 * An optimized version of this random number generator is provided as function
74 * libraries in randoma.zip. These function libraries are coded in assembly
75 * language and support only x86 platforms, including 32-bit and 64-bit
76 * Windows, Linux, BSD, Mac OS-X (Intel based). Use randoma.h from randoma.zip
77 *
78 *
79 * Non-uniform random number generators:
80 * =====================================
81 * Random number generators with various non-uniform distributions are available
82 * in stocc.zip (www.agner.org/random).
