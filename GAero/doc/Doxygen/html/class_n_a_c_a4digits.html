Basic return of coordinates given the genome and the x coordinate.

Genome must contain:

  • genome[0] = 2 first numbers of NACA. i.e. 00 for NACA 0012
  • genome[1] = 2 last numbers. i.e. 12 for NACA 0012. It assumes chord length = 1 unit. yPrevCoor is used to figure out which of the two solutions should be used.

Implements airfoil.
